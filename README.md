# PICO-8 games

Collection of my little pico-8 games' sources.

### Dependencies

* [pico-8](https://www.lexaloffle.com/pico-8.php)

### Useful
* [aseprite](https://www.aseprite.org/) to edit the spritesheet
* [wscode](https://code.visualstudio.com/) to edit the lua files
